Get started with Noodl

Noodl is a low code, node based app builder. It’s a much faster and iterative way to build software than classic code. In Noodl you can use a visual paradigm to build you app, but you can also use code when more convenient. Noodl is also great for designers, as it gives full control over the look and feel of your App. In Noodl everything is live and you edit you app while its live. This site contains all you need to get started!
