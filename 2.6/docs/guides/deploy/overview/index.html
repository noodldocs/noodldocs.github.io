Deploying and hosting Noodl apps

Noodl have it's own hosting infrastructure which makes it very easy to deploy Noodl app. You can also use your own custom domain to point to the app if you have one. If you for some reason want to host the frontend and/or the backend on you own infrastructure that's also possible.
