Data Context Module

This module allows you to work with scoped data objects.

This module is designed to make it easier to work with data objects in complex data scenarios, where the Parent Component Object nodes will not be enough or will make it harder to maintain.

This module is adding 4 new nodes.

  • Context
  • Get State (to get the data from the context, you have to call the Fetch method)
  • Set State
  • Subscriber (this is a reactive object so when something changes on the context this one get the new data automatically)

The Context node is the most important node, where you design the data object you want to use and share it with all the child components. When you have a Context you can use either the Get State or Subscriber nodes (which work similarly to the Variable nodes) to get the current data, it will then find the closest Context node upwards in the graph to get the data.
