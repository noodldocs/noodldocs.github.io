Noodl 2.8 (??)

January 16, 2023

Cloud Functions​

You can now create logic components just like you are used to on the frontend but that run in your Noodl cloud services. This opens up a host of new possibilities that previously required either using an external service like Zapier or writing code, and honestly why would you ever want to leave the comfort of Noodl.

A brand new design​

In 2.8 the editor interface has gone through a re-design to increase consistency and clarity. But most notable the visual component canvas has been replaced with a much nicer preview and the ability to choose which page is currently shown as well as a design mode where UI elements can be selected and revealed in the editor.

Updated version control​

We have a redesigned beautiful new version control system bringing in more of the power features of git to the Noodl environment, with for instance enhanced merging and diffing.

More complete Javascript API​

This release also provides a more complete Javascript API where now anything you can do with a node and connections you can also do with the API (in function and script nodes) for those of you that feel comfortable whipping out your coding skills, go nuts!
