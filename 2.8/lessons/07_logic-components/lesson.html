Logic Components

Logic Components

Just like Visual Components, Logic Components are good for breaking down big node graphs and to make common logic more reusable.

They don't render any visuals so the Logic Component node can be placed freely in node graph like other non-visual nodes.

The Logic components comes with a Component Input node and a Component Output node.

We will make a Logic Component that take the selected Category as an input, and output the correct image and background color.

The States Node

The States Node

A very powerful node in Noodl that we can use to get the right image and color depending on the category, is the States node.

The States node is often used to control UI states with animations, making items hide or appear, and return a specific state with specific values attached.

Creating States & Values

Creating States

You can create multiple states on the States node, that you then can switch between via a Input value. In our case that gonna be the Category port from the Component Input node.

Each state can have multiple value inputs attached which we can set for each state. We are gonna have the Image (string) and Background Color (color) as values, and set the correct image and color for each state.
