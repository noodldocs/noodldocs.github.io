The node graph

The node graph

One of the core concepts of Noodl is the node graph. It's a collection of nodes (think of them like LEGO blocks) and connections between these nodes.

A node represents an element of your prototype, it can be a visual element like a Rectangle or something that is not immediately visible like a logical State. When we put these nodes together with connections it becomes a node graph.

With it we can everything from prototypes to power full web apps that you can deploy directly from Noodl. Check out the graph above, don't worry if it looks complicated now. But this is actually something we will build in this lesson.
