Welcome to Noodl

In this lesson you will learn about the basic concepts of Noodl that combined will let you build powerful web applications.

To get you started we will be building the small counter app you see above, just to introduce you to the core principles of Noodl. One of the greatest things about Noodl is that once you learn these concepts, and how to combine them, there will be virtually no limit to what you can create.

Info

The node graph

The node graph

One of the core concepts of Noodl is the node graph. It's a collection of nodes (think of them like LEGO blocks) and connections between these nodes.

A node represents an element of your app, it can be a visual element like a Rectangle or something that is not immediately visible like a logical Counter When we put these nodes together with connections it becomes a node graph.

With it we can everything from prototypes to power full web apps that you can deploy directly from Noodl. Check out the graph above, don't worry if it looks complicated now. But this is actually something we will build in this lesson.

Visual nodes

Visual nodes

Almost all projects you will be creating have some sort of visual user interface, be it a mobile or desktop app.

For that we use visual nodes, such as images and text. These nodes are blue in color and can be arranged in a hierarchy. What you see now in the node graph editor is a collection of visual nodes. This node graph is the root component, indicated by the small Home icon in the component list to the left. The top node is the root, and everything that is placed under it will be shown in the preview.

You can move nodes by dragging them around with the left mouse button. You can drag a node in and out of the visual hierarchy as you please.

Task

Drag the text node away from the root node

Try dragging the text node away from the hierarchy. As you can see, the text disappears from the preview.

Task

Drag the text node back to make it visible again

You can add a node as a child to another node by dragging it onto that node. Then it will be added as the last child. Try dragging the text node onto the root group. The text will now be visible again in the preview

Task

Move all nodes

You can move all nodes (panning) by pressing the right mouse button and dragging.

Task

Delete the Text nodes

We will not be needing the text node, so delete it. Click it and press delete/backspace.

If you make a mistake when editing you can use Cmd-Z (or Ctrl-Z) to Undo your latest edit.

The node picker

The node picker

Bring up the node picker by clicking the "plus" icon in the top bar, or right click in the empty space of the node graph.

Here you can browse all the core nodes in Noodl and their categories. You can also read about how to use them.

Under each category you can see the corresponding nodes. Click them to place them.

The Group node

The Group node

The Group node is the most basic visual element and is used to structure layouts, group content, and create visual shapes.

If you are familiar with HTML code, a Group is similar to a Div tag in HTML

The sidebar

The sidebar

All nodes have parameters that you can edit to make them behave the way you want. For instance visual nodes have parameters that affect it's appearance, position etc.

If you select a node by left clicking it you will see the parameters of the node in the sidebar to the left.
